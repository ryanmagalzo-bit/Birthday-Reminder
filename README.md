## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔐 **User Authentication** | Secure signup and login with JWT tokens and bcrypt password hashing |
| 📅 **Birthday Management** | Add, edit, delete, and view birthdays with ease |
| ⏱️ **Smart Countdown** | Automatically shows days until next birthday with "Today!" and "Tomorrow!" indicators |
| 🔑 **Password Management** | Change password and forgot password recovery with 6-digit verification codes |
| 🎨 **Beautiful UI** | Glass morphism design with floating animations and cake theme |
| 📱 **Mobile Responsive** | Works perfectly on phones, tablets, and desktops |
| 🔒 **Protected Routes** | All birthday operations require valid JWT authentication |
| 🗄️ **JSON Server Integration** | Lightweight database with full CRUD operations proxied through Express |

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| **Runtime** | Node.js |
| **Web Framework** | Express.js |
| **Data Layer** | JSON Server (via npm package) |
| **Authentication** | bcrypt (password hashing) + JWT |
| **Frontend** | Vanilla JavaScript, HTML5, CSS3 |
| **HTTP Client** | Fetch API |
| **Icons** | Font Awesome 6 |
| **Fonts** | Google Fonts (Inter) |
